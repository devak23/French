Pets - Les Animaux Domestiques (lez-an-ee-mauh domes-ti-eek)
============================================================
the dog = le chien (looh-shian)
the puppy = le chiot (looh-shio)
the cat = le chat (looh-shaa)
the kitten = le chaton (looh-shaa-ton) [n in ton is nasal - as in "taun"]
the fish = le poisson (looh-poi-syon)
the goldfish = le poisson rouge (looh-poi-syon roo-zsh) [n in syon is nasal]
the pig = le cochon (looh-koh-shon) [n in shon is nasal]
the guinea pig = le cochon d'Inde (looh-koh-shon d-an-duh)
the rabbit = le lapin (looh-lap-pan) [n in pan is nasal]
the mouse = le souris (looh-soo-rhee)
the parrot = le perroquet (looh-pero-kay)
the hamster = le hamster (looh-am-sta-ah) ['ah' with a kh sound]
the horse = le cheval (looh-che-vala)
the snake = le serpent (looh-syer-pon) [n in pon is nasal]

the butterfly = le papillon (looh-pah-pi-yon) [n in yon is nasal]
the moth = le papillon de nuit (looh-pah-pi-lon de new-ee)
the beetle = le scarabée (looh-ska-ra-bee)
the centipede = le mille-pattes (looh-mee-lay-pat-h)
the fly = la mouche (lay mooh-sh)
the ladybird = la coccinelle (lah kok-see-nail-luh)
the cockroach = le cafard (looh-kaf-ar)
the dragonfly = la libellule (lah lee-bay-loo-luh)

the scorpion = le scorpion (looh-skor-pee-yon) [n in yon is nasal]
